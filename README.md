tmc_gazebo_plugins
===========================

Overview
===========================
This repository manages the plugin that can be used with any [Gazebo](http://gazebosim.org/) robot.

Plugin
===========================

## mimic_plugin ##
The plugin for interpreting URDF mimic tags.

## ros_hrh_gripper ##
The plugin for High-Ratio-Hypoid gripper.

## ros_bumper ##
The plugin for bumper.
